# NAR-PMI

This repository contains the code for Findings of ACL2021 paper: [A non-autoregressive edit-based approach to controllable text simplification](https://aclanthology.org/2021.findings-acl.330.pdf).

![Editor++](https://user-images.githubusercontent.com/11375341/202476898-891b022a-c1e4-47ab-ae57-2c6dfa6a1ada.jpg)


## Running the models

Scripts to train the models reported in Table 3 can be found in `scripts/run_experiments.sh`.
